# Turkville Dashboard

Turkville haber yönetim paneli - Supabase + Next.js

## Deploy to Vercel

1. GitHub'a push et
2. Vercel'e git ve "Import Project" tıkla
3. Environment variables ekle:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. Deploy!

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
